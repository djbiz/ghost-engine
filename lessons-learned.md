## What I Learned the Hard Way

### Week of 2026-03-27
1. **DM openers that start with "Hey" get ignored** → Switch to observation-based openers (comment on their content specifically)
2. **Don't pitch in the first DM** → Build rapport first, pitch on reply 2-3
3. **Calendly embed needs real URL before testing** → Placeholder URLs break the embed code
4. **Don't send SMS alerts for every lead** → Only alert on: discovery calls booked, Stripe payments, score > 80

### Day of 2026-03-28
1. **FIRST CLOSE — TikTok 45K no-offer creator** → Binary framing + fast execution = 48hr close
2. **Quick Flip as foot-in-door** → Entry product works; upsell path to Full Engine is next
3. **Lead hunt gap** → Need 10-20 fresh leads per day; CRM needs daily fuel
4. **Authority content from case study** → Turn every close into 3 social posts immediately
