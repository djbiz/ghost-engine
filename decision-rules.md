## PHASE 2 RULES — MAGNETIC AUTHORITY ENGINE

### Content Rules

- If posting daily → track every post's DMs + bookings
- If one post drives 3+ inbound → that pillar gets doubled
- If post drives 0 inbound in 48h → kill that pillar, test new angle
- If small win (first sale, first close) → IMMEDIATELY turn into proof post (speed-to-proof.js fires)
- If running 3 ad angles → kill worst performer after 50 impressions
- If cost per booking > $50 → optimize ad copy, not budget
- If cost per booking < $25 → double budget immediately

### Close Rate Rules

- If calls booked but 0 closes → "It's not leads. It's your close." → review Revenue Mirror script
- If lead hesitates on price → apply Money Gap Close
- If lead has audience + NO product → push Quick Flip first ($990, fast win, then upsell)
- If lead has audience + product → push Full Engine ($4,970)
- If lead is creator 50K+ followers + serious energy → push Ghost Partner ($10K setup + $1K/mo)
- If lead says "I need to think" → apply Binary Close: "Think about what? Nothing changes until we fix this."
- If lead says "too expensive" → "Compared to what? You're bleeding X per month right now."

### Deal Stacking Rules

- Every client → attempt Ghost Partner upsell before closing
- Every Ghost Partner → attempt Elite Partner after 60 days
- Every close → run deal-stacker.js to find add-ons

### Momentum Rules

- If 3+ hot leads in 24h → SURGE MODE: faster replies, priority SMS to Derek
- If 2 closes in 48h → MOMENTUM SPIKE: push more outbound, more content, more urgency
- If 0 closes in 7 days → "It's not leads. It's your close." → script review triggered
- If pipeline dry → trigger outbound blast, boost content spend

### Team Rules

- If 10+ calls per week → flag for Setter layer consideration
- If Setter handling replies → Derek only takes qualified calls
- If closer needed → use close-rate-engine.js to train

## THE MONEY MIRROR (Script)

```
Ask: "If we don't fix this... how much do you think you're leaving on the table each month?"

Let them answer.

Say: "So this isn't a cost. This is you plugging that leak."

[They nod]

Say: "The only question is... do we fix it now or keep leaking?"
```

---

## BINARY CLOSE (Script)

```
At end of call:

"We can either build this now... or nothing changes in 90 days. Which one are you choosing?"

[Wait for answer]

If yes → "Cool. I'll send the link now. We start immediately."

If hesitation → "What's making you hesitate? Talk to me."
```

## OBJECTION HANDLER — "Too Expensive"

```

```

## OBJECTION HANDLER — "I Need to Think"
